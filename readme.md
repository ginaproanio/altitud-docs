# ALTITUD Catering

ALTITUD Catering es un servicio gastronómico especializado en ofrecer una experiencia culinaria única con productos nativos ecuatorianos. Con un enfoque en la frescura, autenticidad y creatividad, ALTITUD Catering ofrece menús personalizados para eventos como bodas, fiestas corporativas, cenas privadas y mucho más.

## Estructura del Proyecto

Este repositorio contiene el código fuente para la página web de **ALTITUD Catering**. La página está construida utilizando HTML, CSS y JavaScript, y está diseñada para ser un sitio web estático fácil de navegar.

### Archivos y Carpetas

- **`index.html`**: Página principal del sitio web que presenta los servicios de ALTITUD Catering.
- **`style.css`**: Hoja de estilo para la página web. Contiene todos los estilos y el diseño visual del sitio.
- **`/assets`**: Carpeta donde se almacenarán las imágenes y otros archivos multimedia que se usarán en el sitio.
- **`/js`** (opcional): Carpeta para archivos JavaScript si decides agregar interactividad (por ejemplo, un formulario de contacto dinámico).

### ¿Cómo usar el proyecto?

1. Clona este repositorio a tu máquina local:
    ```bash
    git clone https://github.com/tu-usuario/altitud.git
    ```

2. Navega a la carpeta del proyecto:
    ```bash
    cd altitud
    ```

3. Abre `index.html` en tu navegador.
